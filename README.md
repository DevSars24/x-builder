# X Builder

Internal local-first app for generating, judging, and learning from X post candidates.

## Workspaces

- `engine` - backend, deterministic engine, Codex adapter, persistence.
- `client` - local UI.
- `shared` - shared Zod schemas and TypeScript contracts.
- `e2e-tests` - browser/system tests.
- `tools` - developer tools and local scripts.
- `docs` - product and architecture docs.
